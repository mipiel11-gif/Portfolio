## Strona firmowa – usługi lokalne

Uniwersalna strona internetowa dla lokalnej firmy usługowej,
nastawiona na pozyskiwanie klientów i szybki kontakt.

### Dla kogo
- lokalne firmy usługowe
- jednoosobowe działalności
- małe biznesy

### Rozwiązywane problemy
- brak profesjonalnej strony internetowej
- brak widoczności w internecie
- utrudniony kontakt z klientem

### Funkcje
- sekcja oferty
- dane kontaktowe
- czytelna struktura
- responsywny design

### Technologie
- HTML5
- CSS3

### Testowanie i jakość
- testy responsywności
- testy UX
- sprawdzenie poprawności treści

### Możliwości rozbudowy
- formularz kontaktowy
- system rezerwacji
- backend w Java (Spring Boot)
  
## Możliwości rozbudowy
- panel administracyjny
- system rezerwacji wizyt
- integracja z backendem (Java / Spring Boot)
