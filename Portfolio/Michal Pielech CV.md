## CV online – strona portfolio

Strona internetowa prezentująca CV oraz doświadczenie zawodowe w formie nowoczesnej,
czytelnej wizytówki online.

### Dla kogo
- specjaliści IT
- freelancerzy
- osoby budujące markę osobistą

### Rozwiązywane problemy
- statyczne CV w PDF
- brak nowoczesnej prezentacji umiejętności
- trudność w szybkim zapoznaniu się z profilem kandydata

### Funkcje
- czytelna prezentacja doświadczenia i umiejętności
- responsywny layout
- przejrzysta struktura sekcji
- nowoczesny wygląd

### Technologie
- HTML5
- CSS3

### Testowanie i jakość
- testy czytelności treści
- testy responsywności
- sprawdzenie spójności wizualnej

### Możliwości rozbudowy
- wersja wielojęzyczna
- formularz kontaktowy
- integracja z backendem / API
