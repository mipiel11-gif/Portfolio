# Strona internetowa dla lokalnej firmy

Responsywna i nowoczesna strona internetowa zaprojektowana z myślą o
małych i średnich firmach, które chcą zwiększyć swoją widoczność w internecie
i ułatwić klientom kontakt.

## Dla kogo
- lokalne firmy usługowe
- jednoosobowe działalności gospodarcze
- małe biznesy

## Rozwiązywane problemy
- brak nowoczesnej strony internetowej
- nieczytelna oferta na telefonach
- utrudniony kontakt z klientem
- przestarzały wygląd strony

## Funkcje
- pełna responsywność (mobile / tablet / desktop)
- czytelna struktura i nawigacja
- szybkie ładowanie
- formularz kontaktowy
- podstawowa optymalizacja SEO

## Technologie
- HTML5
- CSS3
- JavaScript
